Social Share par JM Créa vous permet de partager votre contenu sur les réseaux sociaux.

== Description ==

<strong>Social Share est à présent replacé par <a href='https://wordpress.org/plugins/count-share-by-jm-crea/'>Count Share by JM Créa</a></strong>

Social Share par JM Créa vous permet de partager vos posts et/ou articles sur les réseaux sociaux. Ce plugin contient un shortcode.

Vous pouvez définir l’emplacement (haut et/ou bas), choisir le style (horizontal ou vertical) et décider sur quel type de page vous souhaitez l’afficher (page simple et/ou posts).

== Installation ==

1. Uploadez `socialshare` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’SocialShare’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite dans l’onglet ‘Outils -> Social Share’

1. Vous pouvez aussi appeler vos boutons avec le shortcode : [ss_by_jm_crea]
